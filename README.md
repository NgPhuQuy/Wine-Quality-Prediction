# Wine-Quality-Prediction - Đề tài số 7 - Wine Quality Prediction
## Mô tả

Đây là bài toán **dự đoán chất lượng rượu vang** dựa trên các chỉ số hóa học đo được của rượu.  
Mục tiêu của dự án là xây dựng mô hình Machine Learning có khả năng phân tích các đặc trưng này và dự đoán **điểm chất lượng rượu trên thang điểm từ 0 đến 10**.

Dataset được sử dụng là **Wine Quality Dataset** gồm hai loại rượu **Red Wine** và **White Wine**, cung cấp các thuộc tính hóa học như độ axit, lượng đường dư, độ pH, nồng độ cồn, v.v.

- **Nguồn dataset:** UCI Machine Learning Repository  
- **Tên dataset:** Wine Quality (Red + White)  
- **Link:** https://archive.ics.uci.edu/dataset/186/wine+quality  
- **Nguồn thay thế:** Kaggle – "Wine Quality"
